# Test-Retest Brain MRI Datasets — OOD Validation Candidates for Brain-Age

Compiled 2026-04-24 to support OOD validation for a brain-age project. The selection deliberately avoids datasets that are likely already in foundation-model training corpora (HCP, IXI, OASIS, ADNI, UK Biobank, ABCD, CamCAN). Each row was verified against the official data descriptor or repository landing page; any field not directly traceable to a primary source is marked `unverified` rather than guessed.

Notes on rejected / out-of-scope candidates:
- **CAS-PEAL** is a 2D face-recognition dataset, not MRI; excluded as expected.
- **MR-ART** is technically not a multi-visit test-retest dataset (all three runs are acquired in a single visit). It is included because the still vs motion-level pairs can serve as same-day repeat-measure inputs for stability checks, but it does NOT provide a between-day retest interval.
- **NIMH ds004215** has only one MRI session per participant — not test-retest; included for completeness so it does not get re-considered.
- **EDEN2020** appears to be cross-sectional and is non-commercial-only (CC-BY-NC-ND), which limits its utility for a publication that may be funded or commercial.

## Top 5 recommended for OOD test-retest validation

1. **Beijing connectivity-based test-retest (Cao et al., Sci Data 2015) — N=57, 6-week interval, CC-BY-4.0.** This is currently the strongest open candidate. The cohort is large enough for population-level brain-age error analysis (57 subjects x 2 sessions = 114 T1w volumes), the 6-week interval is biologically negligible so any predicted-age delta is true model noise, the Siemens Trio MP-RAGE protocol is standard, and CoRR distribution under CC-BY-4.0 means no DUA friction. The age range 19-30 y is narrow, which is a limitation for absolute brain-age MAE but is exactly the regime where small test-retest deltas are most diagnostic. It is also unlikely to be inside foundation-model pretraining sets that focus on HCP/UKB/IXI/OASIS.

2. **Long-term test-retest (Huang et al., Sci Data 2016) — N=61, 103-189 day interval, CC-BY-4.0.** Complementary to Cao 2015 because the inter-session gap is several months and the second session uses slightly different MPRAGE parameters (1.0 mm iso vs 1.0x1.0x1.33 mm). This makes it a good probe for sensitivity to sequence drift in addition to pure scan-rescan noise. Same INDI/CoRR distribution and license. Also Beijing Normal University, so the two datasets share a scanner family but were collected on different cohorts.

3. **Maclaren et al. (OpenNeuro ds000239) — N=3, 40 scans each, CC0.** Only three subjects, but each is scanned 40 times across 20 days, which gives an exceptionally tight estimate of intra-subject prediction variance. This is the right dataset to report a per-subject standard deviation of predicted brain age. CC0 and OpenNeuro download means zero access friction. Not in any standard foundation-model training set as far as we can verify.

4. **TRCSMM / B-Q MINDED (Zenodo 6473268) — same-day Skyra + PrismaFit cross-scanner, open.** The same-day cross-scanner design isolates pure scanner variance from biological variance, which is a different and complementary axis from days-to-months retest. The headline N (~20) and exact MPRAGE parameters were not parsable from the landing page during this scan and are marked unverified — confirm by downloading the dataset README before relying on it. Two key advantages: it is unlikely to be in foundation-model pretraining (small, recent, niche), and the cross-scanner pairing is a realistic OOD condition for clinical deployment.

5. **GSP retest subsample (Holmes et al. 2015) — N=69, ~77 day interval, MEMPRAGE.** Largest healthy retest subsample of the candidates and good age coverage for the young-adult regime (18-35 y). Caveat: data is behind a Harvard Dataverse data-use agreement (HCP-style), so onboarding latency is non-zero, and the broader GSP cohort is sometimes included in pretraining corpora — **verify your foundation model's training manifest before using GSP for OOD claims.** If the model card is silent on GSP, treat it as potentially seen rather than truly OOD.
